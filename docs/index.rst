simGL
=====

**simGL** simulates genotype likelihoods (GLs) from haplotypic genotype matrices,
given information about per-sample coverage and sequencing error rates.
It is designed to work seamlessly with the output of
`tskit <https://tskit.dev>`_ and `msprime <https://tskit.dev/msprime/>`_
in Python pipelines, but can be used with any haplotypic genotype matrix.
